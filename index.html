Acerca de mi

Nico Marshal

Desarrollador .NET

¡Hola! Me llamo Nicolás Ledesma, vivo en la ciudad de Córdoba, Argentina y soy desarrollador .NET.

En mis inicios comencé desarrollando diferentes proyectos Frontend, como interfaces o componentes reutilizables, con el objetivo de afianzar e integrar los conocimientos que había adquirido de manera autodidacta, como HTML, CSS, JavaScript, AJAX, Progressive Web Apps, Git & GitHub y una gran variedad de APIs, tanto nativas como no nativas.

Posteriormente, empecé a interesarme muchísimo en lo que respecta al desarrollo Backend, por ende, decidí realizar un curso de C# .NET de 7 meses de duración dictado por el Polo Tecnológico de Mina Clavero junto con la empresa IncluIT

Finalizar ese curso fue el principio de toda esta aventura en .NET

Luego de algunas semanas, quedé seleccionado para participar en "IT Bootcamp .NET eFALCOM by Alkemy": un Bootcamp de 6 semanas certificado por Alkemy donde vimos en profundidad cada uno de los proyectos que se pueden realizar con ASP.NET.

En las últimas 2 semanas del Bootcamp, como proyecto final, realizamos la API de una Billetera Virtual llamada "eWallet", la cual nos permite realizar transacciones entre usuarios y más funcionalidades

APLICACIÓN DE CHAT

Interfaz responsiva de un sistema de mensajería que fue maquetada con HTML, CSS, Flexbox CSS y Grid CSS y, programada con JavaScript.

Se desarrollaron funcionalidades tales como enviar (o recibir) y almacenar mensajes, acceder a la cámara, obtener la ubicación actual, cargar documentos e imágenes, usar emojis, previsualizar imágenes con sliders, entre otras.

CARTÓN DE BINGO

Aplicación maquetada con HTML, CSS, Grid CSS y, programada con JavaScript.

Es un cartón de bingo aleatorio que debe cumplir determinadas condiciones.
Por ejemplo: "Cada columna debe tener uno o dos números"

Para resolverlo, se utilizaron diferentes herramientas matemáticas tales como vectores y matrices

BILLETERA VIRTUAL "eWALLET API"

Desarrollamos una API en C# con .NET en la cual el usuario puede realizar diferentes funcionalidades tales como iniciar sesión, depositar dinero en su cuenta, transferir a otras cuentas, actualizar sus datos, entre otras.

También generamos un rol de administrador para aquellos usuarios autorizados a realizar tareas administrativas y de mantenimiento sobre los otros usuarios.
